# Save Session Context

Summarize everything accomplished during this session and update PROJECT_CONTEXT.md:

1. Review the entire conversation history from this session
2. Identify all significant changes, decisions, and implementations made:
   - Files created, modified, or deleted
   - Features added or bugs fixed
   - Architecture decisions or pattern changes
   - Configuration updates
   - New insights or learnings about the codebase

3. Read the existing `PROJECT_CONTEXT.md` file to understand its current structure

4. Create a well-organized summary of this session's work including:
   - Date/timestamp
   - What was accomplished
   - Key technical decisions
   - Files affected
   - Any new patterns or conventions established

5. Patch the summary into `PROJECT_CONTEXT.md` in an appropriate location (either append to existing section or create new dated entry)

6. Ensure the update maintains the document's existing format and organization

Do not create a todo list for this command - focus on accurately capturing and preserving the session's work.
